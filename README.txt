This is the GitHub repository for the Master's thesis of Kasper Porsborg Andersen.
The folder structure is not intuitive, so here will be mentioned a few of the more relevant code scripts 
that were used in the production of the figures in said Master's thesis.

Code_March/Rabi_oscillations.py		--- Does indeed contain Rabi oscillations but also the Hahn echo
sequence and inversion recovery sequence

Code_January/New_CW-EPR.py		--- Contains the CW-EPR plots. Note it still requires manual 
changes to achieve the different versions of Clar' goblet and the naphthalene radical

Code_January/fitParameterFactory.py	--- Contains the calculations of the generic fit parameters

Code_December/Fit_Bleaney-Bower.py	--- Contains the Bleaney-Bowers plot

Code_September/Optical_Properties.py	--- Contains the calculations of the polarisability
